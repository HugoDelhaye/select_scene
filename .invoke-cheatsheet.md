# Invoke Tasks Quick Reference

## Setup & Installation

```bash
invoke setup                  # Install Python dependencies
invoke ensure-dirs            # Ensure all required directories exist
```

## Data Processing

```bash
invoke process-data           # Generate metrics from raw data
invoke process-ppo-variables  # Extract PPO variables from JSON
```

## Data Conversion

```bash
invoke convert-csv-to-parquet    # CSV → Parquet
invoke convert-parquet-to-csv    # Parquet → CSV
```

## Visualization & Dashboard

```bash
invoke dashboard              # Launch web dashboard (http://localhost:8050)
```

## Export to HTML

```bash
invoke export-html                          # Export all scenes
invoke export-html --level=w1l1             # Export all from level w1l1
invoke export-html --level=w1l1 --scene=0   # Export single scene
```

## Data Exploration

```bash
invoke stats                  # Display dataset statistics
invoke list-scenes            # List all available scenes
```

## Cleanup

```bash
invoke clean                  # Remove generated HTML outputs
```

## Complete Pipeline

```bash
invoke run                    # Process data + launch dashboard
```

## General Commands

```bash
invoke --list                 # Show all available tasks
invoke --help <task>          # Show help for specific task
```

## Configuration

All settings are in `invoke.yaml`:
- Directory paths
- Data file locations
- Dashboard settings (host, port)
- Export configurations

## Task Definition

Tasks are defined in `tasks.py`. You can add custom tasks by creating new functions with the `@task` decorator.
